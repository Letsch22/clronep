Provide creation and usage information of specified resource according to the specified Options (eg, Options could specify to only return the Description).

public Result info (string clientkey, string rid, object options)

⊟ Parameters

clientkey
The client identification key.
rid
The resource id.
options
Each boolean entry defaults to false. If an empty Options list is given, then all are set to true. Timestamps default to the start of time or the end of time depending on their context.
  • Not all resource types have the same set of Options.
